# [Wio Personal - Send: Recurring Payments](https://app.getguru.com/card/caxBG9ai/Wio-Personal-Send-Recurring-Payments)

<p class="ghq-card-content__paragraph ghq-is-empty" data-ghq-card-content-type="paragraph">
</p>
<p class="ghq-card-content__paragraph" data-ghq-card-content-type="paragraph">
 <strong class="ghq-card-content__bold" data-ghq-card-content-type="BOLD">
  Recurring payments:
 </strong>
 provide customers with the option of scheduling their periodic payments and transfers from the application itself so that they can cater to their recurring payments without any intervention.
</p>
<p class="ghq-card-content__paragraph ghq-is-empty" data-ghq-card-content-type="paragraph">
 <strong class="ghq-card-content__bold" data-ghq-card-content-type="BOLD">
 </strong>
</p>
<p class="ghq-card-content__paragraph" data-ghq-card-content-type="paragraph">
 <strong class="ghq-card-content__bold" data-ghq-card-content-type="BOLD">
  Key Points:
 </strong>
</p>
<ol class="ghq-card-content__numbered-list" data-ghq-card-content-type="NUMBERED_LIST" start="1">
 <li class="ghq-card-content__numbered-list-item" data-ghq-card-content-type="NUMBERED_LIST_ITEM">
  Customer can choose to initiate recurring transfer to other bank accounts.
 </li>
 <li class="ghq-card-content__numbered-list-item" data-ghq-card-content-type="NUMBERED_LIST_ITEM">
  For now, the feature is only applicable for local transfer.
 </li>
 <li class="ghq-card-content__numbered-list-item" data-ghq-card-content-type="NUMBERED_LIST_ITEM">
  Customer can choose the frequency of the recurring transfer (Once-Daily-Weekly-Monthly)
 </li>
 <ul class="ghq-card-content__bulleted-list" data-ghq-card-content-type="BULLETED_LIST">
  <li class="ghq-card-content__bulleted-list-item" data-ghq-card-content-type="BULLETED_LIST_ITEM">
   Once - Customer can choose the date
  </li>
  <li class="ghq-card-content__bulleted-list-item" data-ghq-card-content-type="BULLETED_LIST_ITEM">
   Daily - Customer can choose start &amp; end date
  </li>
  <li class="ghq-card-content__bulleted-list-item" data-ghq-card-content-type="BULLETED_LIST_ITEM">
   Weekly - Customer can choose start &amp; end date
  </li>
  <li class="ghq-card-content__bulleted-list-item" data-ghq-card-content-type="BULLETED_LIST_ITEM">
   Monthly - Customer can choose start &amp; end date
  </li>
 </ul>
 <li class="ghq-card-content__numbered-list-item" data-ghq-card-content-type="NUMBERED_LIST_ITEM">
  Transactions would occur after 10 AM starting the defined date according to the previous setup.
  <strong class="ghq-card-content__bold" data-ghq-card-content-type="BOLD">
   (Monday to Saturday)
  </strong>
 </li>
 <li class="ghq-card-content__numbered-list-item" data-ghq-card-content-type="NUMBERED_LIST_ITEM">
  On bank holidays, transfer would occur next working day.
 </li>
 <li class="ghq-card-content__numbered-list-item" data-ghq-card-content-type="NUMBERED_LIST_ITEM">
  Transfer will be pending for the defined date, amount won't be debited from customer account
  <strong class="ghq-card-content__bold" data-ghq-card-content-type="BOLD">
   (Won't be on hold)
  </strong>
 </li>
 <li class="ghq-card-content__numbered-list-item" data-ghq-card-content-type="NUMBERED_LIST_ITEM">
  Customer can cancel the transfer anytime before 10 AM of the defined date.
 </li>
</ol>
<p class="ghq-card-content__paragraph" data-ghq-card-content-type="paragraph">
 <strong class="ghq-card-content__bold" data-ghq-card-content-type="BOLD">
  Reminder
 </strong>
 : Customer can also setup recurring transfer from current account to savings space
</p>

<h1 class="ghq-card-content__large-heading" data-ghq-card-content-type="LARGE_HEADING">
 FAQs
</h1>
<p class="ghq-card-content__paragraph align-left" data-ghq-card-content-type="paragraph" data-text-align="left">
 <strong class="ghq-card-content__bold" data-ghq-card-content-type="BOLD">
  <span class="ghq-card-content__text-color" data-ghq-card-content-type="TEXT_COLOR" style="color:#9013fe">
   How do I setup recurring transfer?
  </span>
 </strong>
</p>
<p class="ghq-card-content__paragraph" data-ghq-card-content-type="paragraph">
 To set up a recurring transfer, log in and go to the 'Scheduled' section next to 'Recent' section in Send Dashboard. Click “Add Recurring Transfer,” enter the amount, frequency, start date, recipient’s info, and confirm the Scheduler will be created. To convert a regular local transfer to recurring, start a normal AED transfer to UAE and toggle “Scheduled Transfer” on the amount entry screen. You can view or Cancel your scheduled transfers from the same 'Scheduled' section.
</p>
<p class="ghq-card-content__paragraph ghq-is-empty" data-ghq-card-content-type="paragraph">
</p>
<p class="ghq-card-content__paragraph" data-ghq-card-content-type="paragraph">
 <strong class="ghq-card-content__bold" data-ghq-card-content-type="BOLD">
  <span class="ghq-card-content__text-color" data-ghq-card-content-type="TEXT_COLOR" style="color:#9013fe">
   What are my options for setting up recurring transfer?
  </span>
 </strong>
</p>
<p class="ghq-card-content__paragraph" data-ghq-card-content-type="paragraph">
 In the Send dashboard, customers have several options for setting up a recurring transfer:
</p>
<ul class="ghq-card-content__bulleted-list" data-ghq-card-content-type="BULLETED_LIST">
 <li class="ghq-card-content__bulleted-list-item" data-ghq-card-content-type="BULLETED_LIST_ITEM">
  <strong class="ghq-card-content__bold" data-ghq-card-content-type="BOLD">
   Once:
  </strong>
  Schedule a single future transfer.
 </li>
 <li class="ghq-card-content__bulleted-list-item" data-ghq-card-content-type="BULLETED_LIST_ITEM">
  <strong class="ghq-card-content__bold" data-ghq-card-content-type="BOLD">
   Daily:
  </strong>
  Set up transfers to occur every day.
 </li>
 <li class="ghq-card-content__bulleted-list-item" data-ghq-card-content-type="BULLETED_LIST_ITEM">
  <strong class="ghq-card-content__bold" data-ghq-card-content-type="BOLD">
   Weekly:
  </strong>
  Schedule transfers to happen once a week.
 </li>
 <li class="ghq-card-content__bulleted-list-item" data-ghq-card-content-type="BULLETED_LIST_ITEM">
  <strong class="ghq-card-content__bold" data-ghq-card-content-type="BOLD">
   Monthly:
  </strong>
  Arrange for transfers to take place once a month.
 </li>
</ul>
<p class="ghq-card-content__paragraph" data-ghq-card-content-type="paragraph">
 Additionally, you can select both the start date and end date for their recurring transfers. These options can be set while setting up a Scheduled transfer.
</p>
<p class="ghq-card-content__paragraph ghq-is-empty" data-ghq-card-content-type="paragraph">
