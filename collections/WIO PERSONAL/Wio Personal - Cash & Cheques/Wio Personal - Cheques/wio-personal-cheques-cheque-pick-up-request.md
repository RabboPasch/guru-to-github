<h1 class="ghq-card-content__large-heading" data-ghq-card-content-type="LARGE_HEADING">
 Guidelines for Cheque Pick-Up Services: Eligibility, Requirements, and Turnaround Times
</h1>
<ul class="ghq-card-content__bulleted-list" data-ghq-card-content-type="BULLETED_LIST">
 <li class="ghq-card-content__bulleted-list-item" data-ghq-card-content-type="BULLETED_LIST_ITEM">
  <strong class="ghq-card-content__bold" data-ghq-card-content-type="BOLD">
   Eligibility Criteria:
  </strong>
 </li>
 <ul class="ghq-card-content__bulleted-list" data-ghq-card-content-type="BULLETED_LIST">
  <li class="ghq-card-content__bulleted-list-item" data-ghq-card-content-type="BULLETED_LIST_ITEM">
   Customer can hand up to 20 cheques per visit.
  </li>
  <li class="ghq-card-content__bulleted-list-item" data-ghq-card-content-type="BULLETED_LIST_ITEM">
   Maximum of
   <strong class="ghq-card-content__bold" data-ghq-card-content-type="BOLD">
    2 visits allowed per customer per month
   </strong>
  </li>
 </ul>
 <li class="ghq-card-content__bulleted-list-item" data-ghq-card-content-type="BULLETED_LIST_ITEM">
  <strong class="ghq-card-content__bold" data-ghq-card-content-type="BOLD">
   Customer:
  </strong>
 </li>
 <ul class="ghq-card-content__bulleted-list" data-ghq-card-content-type="BULLETED_LIST">
  <li class="ghq-card-content__bulleted-list-item" data-ghq-card-content-type="BULLETED_LIST_ITEM">
   Should write the deposit account number on the
   <strong class="ghq-card-content__bold" data-ghq-card-content-type="BOLD">
    back
   </strong>
   of the cheque
  </li>
  <li class="ghq-card-content__bulleted-list-item" data-ghq-card-content-type="BULLETED_LIST_ITEM">
   Customer does not need to
   <strong class="ghq-card-content__bold" data-ghq-card-content-type="BOLD">
    scan the cheque
   </strong>
  </li>
  <li class="ghq-card-content__bulleted-list-item" data-ghq-card-content-type="BULLETED_LIST_ITEM">
   Signature should
   <strong class="ghq-card-content__bold" data-ghq-card-content-type="BOLD">
    match their Emirates ID
   </strong>
   signature or as per customer profile
  </li>
  <li class="ghq-card-content__bulleted-list-item" data-ghq-card-content-type="BULLETED_LIST_ITEM">
   Cheque collection doesn't necessarily have to be collected from the authorized signatory.
  </li>
 </ul>
 <li class="ghq-card-content__bulleted-list-item" data-ghq-card-content-type="BULLETED_LIST_ITEM">
  <strong class="ghq-card-content__bold" data-ghq-card-content-type="BOLD">
   TransGuard:
  </strong>
 </li>
 <ul class="ghq-card-content__bulleted-list" data-ghq-card-content-type="BULLETED_LIST">
  <li class="ghq-card-content__bulleted-list-item" data-ghq-card-content-type="BULLETED_LIST_ITEM">
   TG will give the customer an acknowledgment receipt by the time of cheque pick up
  </li>
  <li class="ghq-card-content__bulleted-list-item" data-ghq-card-content-type="BULLETED_LIST_ITEM">
   <strong class="ghq-card-content__bold" data-ghq-card-content-type="BOLD">
    TransGuard TAT for Cheque Pick Up Services:
   </strong>
  </li>
  <ul class="ghq-card-content__bulleted-list" data-ghq-card-content-type="BULLETED_LIST">
   <li class="ghq-card-content__bulleted-list-item" data-ghq-card-content-type="BULLETED_LIST_ITEM">
    T+1  (next day) cut off is 6pm
   </li>
   <li class="ghq-card-content__bulleted-list-item" data-ghq-card-content-type="BULLETED_LIST_ITEM">
    T+2 - For negative areas &amp; outskirts of city limits (2 days from request day) cut off is 6pm
   </li>
   <li class="ghq-card-content__bulleted-list-item" data-ghq-card-content-type="BULLETED_LIST_ITEM">
    <strong class="ghq-card-content__bold" data-ghq-card-content-type="BOLD">
     Total SLA:
    </strong>
    5 business days
   </li>
  </ul>
 </ul>
</ul>
